## üöÄ Quick Start

### Explore via Memory Workspace:
1. Open http://localhost:5173
2. Go to Memory Fusion panel
3. Navigate folders in left panel
4. Click files to view content
5. Edit, save, create, upload files

### Programmatic Access:
```python
# List all training files
GET /api/memory/files

# Read specific file
GET /api/memory/file?path=documents/company_vision_2025.md

# Create new training file
POST /api/memory/file?path=new_doc.md&content=...
```
