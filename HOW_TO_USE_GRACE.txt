====================================================================
HOW TO USE GRACE - COMPLETE GUIDE
====================================================================

STEP 1: Start Ollama (Required for LLMs)
-----------------------------------------
In a terminal:
  ollama serve

Keep this running.


STEP 2: Install Models (One Time)
----------------------------------
In another terminal:
  scripts\startup\install_all_models.cmd

This installs all 15 open source models (~313GB, 2-3 hours).

Or install just essentials:
  ollama pull qwen2.5:32b
  ollama pull deepseek-coder-v2:16b
  ollama pull kimi:latest


STEP 3: Start Grace
-------------------
  python serve.py

Or:
  START.cmd

Grace will:
  - Find available port
  - Check which models are installed
  - Start API server
  - Show you the URL


STEP 4: Use Grace
-----------------

Option A: Remote Access
  python remote_access_client.py setup
  python remote_access_client.py shell

Option B: Autonomous Learning
  python start_grace_now.py

Option C: Interactive Menu
  USE_GRACE.cmd

FEATURES
====================================================================

Remote Access:
  - Zero-trust device authentication
  - Multi-factor authentication
  - RBAC with 5 roles
  - Complete session recording
  - WebSocket real-time shell

Autonomous Learning:
  - 11 knowledge domains
  - 25+ learning projects
  - Builds: CRM, E-commerce Analytics, Cloud Infrastructure
  - Sandbox experimentation
  - KPI tracking & trust scores

Open Source LLMs (15 models):
  - Conversation: qwen2.5 (32B, 72B)
  - Coding: deepseek-coder, granite-code, codegemma
  - Reasoning: deepseek-r1 (o1-level)
  - Vision: llava (sees images)
  - Fast: phi3.5, gemma2, llama3.2
  - All 100% free and private!
