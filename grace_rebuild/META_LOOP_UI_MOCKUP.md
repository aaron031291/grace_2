## Interactive Elements

### Recommendation Card - Hover States
```
Normal:      border: 2px solid rgba(risk_color, 0.2)
Hover:       border: 2px solid rgba(risk_color, 0.6)
             box-shadow: 0 4px 12px rgba(risk_color, 0.3)
```

### Buttons
```
Approve:     background: #00ff88
             color: #000
             hover: brightness(1.1)

Reject:      background: #ff4444
             color: #fff
             hover: brightness(1.1)

Rollback:    background: #ff4444
             color: #fff
             hover: brightness(1.1)
```

### Charts
```
Effectiveness: Purple bars (#7b2cbf)
Trends:        Cyan line (#00d4ff)
Hover:         Tooltip with timestamp and value
```

## Mobile Responsive Breakpoints

```
Desktop (> 1024px):  3-column layout, full charts
Tablet (768-1024px): 2-column layout, simplified charts
Mobile (< 768px):    1-column stack, mini charts
```

```

3. Badge appears: "NEW"
4. Remove "NEW" badge after 3 seconds
```
