# Legacy Code Archive

This folder contains deprecated UI components that are not part of the active build.

These files are excluded from the TypeScript compilation via `tsconfig.app.json`.

## Active Components (in production)
- **AppChat.tsx** - Main chat interface with sidebar controls
- **ChatPanel.tsx** - Core chat component
- **FileExplorer.tsx** - Memory file browser
- **BackgroundTasksDrawer.tsx** - Task management
- **RemoteCockpit.tsx** - Remote access controls
- **HealthMeter.tsx** - System health monitoring

## Legacy Components (archived here)
- Multiple experimental Grace UI variations
- Various prototype App components
- Old chat implementations
- Deprecated API integrations
