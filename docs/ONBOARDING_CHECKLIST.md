# Onboarding Checklist

- [ ] Gain repository access and copy `.env.example`.
- [ ] Generate JWT secret and configure local `.env`.
- [ ] Install backend + frontend dependencies.
- [ ] Run `py minimal_backend.py` and verify `/health`.
- [ ] Log into frontend (create admin user if needed).
- [ ] Read `docs/CONTRIBUTOR_GUIDE.md` and `docs/CHANGE_MANAGEMENT.md`.
- [ ] Review domain docs relevant to your role.
- [ ] Pair with existing contributor for first PR.

Store completed checklists in the team's knowledge base.
