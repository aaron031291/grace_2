## System Architecture

### Two-Layer Design

**Layer 1: Unbreakable Core (Always Running)**
- 16 kernels managed by control plane
- Message bus for kernel communication
- Immutable audit log
- Clarity kernel (component registry)
- Auto-restart on failures
- **Never stops**, even if API crashes

**Layer 2: FastAPI (External Interface)**
- HTTP/WebSocket endpoints
- User interface backend
- Can crash without affecting core
- Auto-restarts via control plane

```bash
# Ensure backend is running
```

