### API Endpoints:

**Get Status:**
```bash
curl http://localhost:8000/api/autonomous/improver/status
```

**Trigger Manual Scan:**
```bash
curl -X POST http://localhost:8000/api/autonomous/improver/trigger
```

**Change Interval:**
```bash
curl -X PATCH http://localhost:8000/api/autonomous/improver/config \
  -H "Content-Type: application/json" \
  -d '{"scan_interval": 600}'  # 10 minutes
```

**Start/Stop:**
```bash
curl -X POST http://localhost:8000/api/autonomous/improver/start
curl -X POST http://localhost:8000/api/autonomous/improver/stop
```
