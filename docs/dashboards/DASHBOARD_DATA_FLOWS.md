## Error Handling Map

### API Error Codes & UI Responses

| **HTTP Code** | **Error Type** | **UI Response** | **User Action** |
|---------------|----------------|-----------------|-----------------|
| 400 | Bad Request | Toast: "Invalid input" | Fix form, retry |
| 401 | Unauthorized | Redirect to login | Re-authenticate |
| 403 | Forbidden | Toast: "Permission denied" | Contact admin |
| 404 | Not Found | Toast: "Resource not found" | Refresh page |
| 409 | Conflict | Modal: "Duplicate exists" | Choose different name |
| 429 | Rate Limited | Toast: "Too many requests, wait 30s" | Auto-retry after delay |
| 500 | Server Error | Modal: "Server error, try again" | Retry or contact support |
| 503 | Service Unavailable | Banner: "Service under maintenance" | Wait and retry |

### Network Error Handling

```typescript
try {
  const response = await axios.get('/api/telemetry/...');
  // Success path
} catch (error) {
  if (error.response) {
    // Server responded with error status
    handleAPIError(error.response.status, error.response.data);
  } else if (error.request) {
    // Request made but no response (network issue)
    showNetworkErrorBanner("Unable to reach server");
  } else {
    // Request setup error
    showGenericErrorToast("An unexpected error occurred");
  }
}
```
